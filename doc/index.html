Preface

This is the reference manual of lua-nanovg, which is a Lua binding library for NanoVG and for NanoSVG. [1]

It is assumed that the reader is familiar with NanoVG, NanoSVG, GLFW(through moonglfw) and the Lua programming language.

For convenience of reference, this document contains external (deep) links to the Lua Reference Manual and the GLFW documentation.

Getting and installing

For installation intructions, refer to the README file in the lua-nanovg official repository on GitHub.

Module organization

The lua-nanovg module is loaded using Lua’s require() and returns a table containing the functions it provides (as usual with Lua modules). This manual assumes that such table is named nvg, i.e. that it is loaded with:

 local nvg = require("nvg")

but nothing forbids the use of a different name.

Examples

A few examples can be found in the examples/ directory of the release package.

License

lua-nanovg is released under the MIT/X11 license (same as Lua, and with the same only requirement to give proper credits to the original author). The copyright notice is in the LICENSE file in the base directory of the official repository on GitHub.

Bindings

The functions list of the nvg module

To use lua-nanovg, you should first require it, after that, you will get a function to create context for drawing:

 local ctx = require("nvg")

You can pass some arguments to this function, to specify the atlas size and/or edge AA (anti-alias) flags:

 local ctx = require 'nvg' (256, 256, "AA") -- default
 local ctx = require 'nvg' (256, 256) -- no edge anti-alias
 local ctx = require 'nvg' "AA" -- use edge anti-alias, and default atlas size.

After you get context, you can draw with it. All APIs will exactly same with HTML5 canvas.

 ctx:beginFrame()
 ctx:endFrame()
 ctx:color("name")
 ctx:color("#rgb")
 ctx:color("#argb")
 ctx:color("#rrggbb")
 ctx:color("#aarrggbb")
 ctx:color(r,g,b)
 ctx:color(r,g,b,a)
 ctx:color("HSL", h,s,l);
 ctx:color("HSL", h,s,l,a);

 ctx:save()
 ctx:restore()
 ctx:reset()

 ctx.strokeStyle = color|paint;
 ctx.fillStyle = color|paint;

 ctx.lineCap = "butt|round|square";
 ctx.lineJoin = "round|bevel|miter";
 ctx.lineWidth = width;
 ctx.miterLimit = number;

 ctx:resetTransform()
 ctx:scale()
 ctx:rotate()
 ctx:translate()
 ctx:transform()
 ctx:setTransform()


 ctx:image() -> ctx.image.load
 ctx:drawImage()
 ctx.image.new()
 ctx.image.load()
 =image.width
 =image.height
 image:update()
 image:delete()

 ctx:gradient()
 ctx.gradient.linear()
 ctx.gradient.box()
 ctx.gradient.radial()

 ctx:scissor()

 ctx.pathWnding = "(solid|hole)|(CCW|CW)";
 ctx:beginPath()
 ctx:moveTo()
 ctx:lineTo()
 ctx:bezierTo() == ctx:bezierCurveTo()
 ctx:arcTo()
 ctx:closePath()

 ctx:arc()
 ctx:rect()
 ctx:roundRect()
 ctx:ellipse()
 ctx:circle()
 ctx:fill()
 ctx:stroke()

 ctx.font = font;
 ctx.textAlign =
 ctx.fontBlur = number;
 ctx:text()
 ctx:measureText()

NanoVG.Context

Context creation and manipulation

  • The NanoVG.Context table contains the following version fields:

    • _VERSION: a string describing the NanoVG version

  • inst = new( ) Context constructor.

  • font( ) Font setter.

  • linearGradient(sx, sy, ex, ey, iColor, oColor) Defines a linear gradient paint.

  • boxGradient(x, y, w, h, r, f, iColor, oColor) Defines a rectangular gradient paint.

  • radialGradient(cx, cy, inr, outr, iColor, oColor) Defines a radial gradient paint.

  • clear(color) Clears the context using the specified color { r=0, g=0, b=0, a=0 }.

  • beginFrame() Starts a frame scope.

  • cancelFrame() Cancels a frame scope.

  • endFrame() Flushes a frame scope.

  • save() Saves the context.

  • restore() Restores the context.

  • reset() Resets the context.

  • a, b, c, d, e, f = currentTransform() Getter for the current transformation.

  • transform(a, b, c, d, e, f) Setter for the current transformation.

  • translate(x, y) Translate by x horizontally and y vertically.

  • scale(x, y) Scale with x horizontally and y vertically.

  • rotate(angle) Rotate by angle.

  • skew(x, y) Skew with x horizontally and y vertically.

  • resetScissor() Resets the scissor.

  • scissor(x, y, w, h) Applies the scissor.

  • intersectScissor(x, y, w, h) Applies the intersection scissor.

  • globalCompositeOperation(enum:CompositeOperation) Sets the composite operation.

  • globalCompositeBlendFunc(enum:BlendFactor) Sets the composite blending function operation.

  • globalCompositeBlendFuncSeparate(enum:BlendFactor) Sets the composite blending function operation.

  • beginPath() Starts a new path.

  • moveTo(x, y) Moves the drawing head.

  • lineTo(x, y) Draws a line from the head to the specified coordinates.

  • quadraticCurveTo(cx, cy, x, y) Draws a quadratic curve from the head to the specified coordinates.

  • bezierCurveTo(c1x, c1y, c2x, c2y, x, y) Draws a bezier curve from the head to the specified coordinates.

  • arcTo(x1, y1, x2, y2, radius) Draws an arc from the head to the specified coordinates.

  • closePath() Closes the previously opened path.

  • arc(cx, cy, r, a0, a1, dir:ArcDirection) Draws an arc from the head to the specified coordinates.

  • rect(x, y, w, h) Draws a rect from the head to the specified coordinates.

  • roundedRect(x, y, w, h, r1|, r2, r3, r4) Draws a rounded rectangle defined by the coordinates.

  • ellipse(cx, cy, rx, ry) Draws an ellipse defined by the coordinates.

  • circle(cx, cy, r) Draws a circle defined by the coordinates.

  • fill() TODO.

  • stroke() TODO.

  • text(x, y, text) Draws the text.

  • textMetrics(ascender, descender, lineHeight) Sets the text metrics.

  • textBounds(string|(x, y|, width)) Sets the text bounds.

  • addFallbackFont(font, fallbackFont) Sets an alternate font for the main font.

  • shapeAntiAlias(flag) Enable of disable the antialiasing of shapes.

  • globalAlpha(a) Sets the global scene alpha opacity.

  • strokeColor(color:Color) Sets the stroke color.

  • fillColor(color:Color) Sets the fill color.

  • strokeStyle(color:Color|paint:Paint) Sets the stroke style using a color or a paint.

  • fillStyle(color:Color|paint:Paint) Sets the fill style using a color or a paint.

  • miterLimit(width) Sets the miter width limit.

  • lineWidth(width) Sets the width of the line.

  • lineCap(type:{butt|round|square}) Sets the line cap type.

  • lineJoin(type:{round|bevel|miter}) Sets the line join type.

  • pathWinding(winding:PathWinding) Sets the path winding.

  • fontFace(face) Sets the font face(family).

  • fontSize(size) Sets the dimensions of the font.

  • fontBlur(blur) Sets the amount of blur.

  • textAlign(align:Align) Sets the text alignment.

  • textLetterSpacing(space) Sets the width of the space between letters.

  • textLineHeight(height) Sets the text line height.

NanoVG.Color

Color creation and manipulation

  • NanoVG.Color attributes:

    • r Reader for the red component.

    • g Reader for the green component.

    • b Reader for the blue component.

    • a Reader for the alpha(opacity) component.

  • parse(input:Number|String) Tries to parse input to a valid color.

  • color = rgb(r,g,b,a) Creates a color with specified components.

  • color = rgb8(r,g,b,a) Creates a color with specified components.

  • color = hsl(h,s,l,a) Creates a color with specified components.

  • color = lerp(c0,c1,u) Creates a color with specified components.

NanoVG.Image

Image creation and manipulation

  • NanoVG.image attributes:

    • ox Reader for ox coordinate.

    • oy Reader for oy coordinate.

    • ex Reader for ex coordinate.

    • ey Reader for ey coordinate.

    • angle Reader for angle coordinate.

    • alpha Reader for alpha property.

    • image Reader for image pointer.

    • repeat Flag if image is to be repeated.

  • load(path, flags:ImageFlags) Creates and loads an image from specified path.

  • data(data, flags:ImageFlags) Creates and loads an image from data.

  • rgba(w, h, flags:ImageFlags, data:string) Creates and loads a rgba image from data.

  • update(data:string) Updates image with the bytes specified by data.

  • w, h = size() Reads the image size.

  • extent(ox, oy, ex, ey) Sets the image extent.

  • width() Reads the image width.

  • height() Reads the image height.
